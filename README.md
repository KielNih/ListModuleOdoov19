Odoo 19 Custom Modules Portfolio  
This repository contains a collection of advanced, production-ready custom modules developed for Odoo 19.0 using Python 3.10. These modules are designed to solve 
critical business gaps in Inventory Management, Quality Control, and Human Resources by leveraging Odoo's modern framework components like OWL (Odoo Web Library), 
Scheduled Actions (ir.cron), and the automated Activity Mixin.

🛠️ Modules Overview
1. Auto-Checklist for Goods Received (stock_qc_checklist)  
Objective: Ensures warehouse accountability and prevents forgotten quality inspections during vendor stock receipts.

2. Stock Threshold & Dynamic Dashboard (custom_stock_dashboard)  
Objective: Provides real-time visual dynamic monitoring of critical stock levels directly on the Odoo dashboard.

3. HR Document Expiry Tracker (hr_document_expiry_tracker)  
Objective: Proactively manages compliance and tracking of critical employee documentation (e.g., Contracts, ID Cards, Certifications).
